<h2> Print basic array </h2>


```java
import java.util.*;
class array2d{
  public static void main(String [] args){
  
    int[] arr={1,2,3,4,5};
    
    for(int i=0;i<arr.length;i++){
      System.out.println(arr[i]);
    }
  }
}
```
